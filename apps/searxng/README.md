# SearXNG

SearXNG 是一个免费的互联网元搜索引擎，聚合来自多个搜索服务的结果。它不存储用户搜索信息，不跟踪用户行为，完全保护用户隐私。部署完成后可直接通过浏览器访问。

## 特性

* **隐私保护**：不跟踪用户，不存储搜索记录
* **元搜索**：聚合多个搜索引擎的结果
* **开源免费**：完全开源，可自由部署
* **多语言支持**：支持多种界面语言
* **可定制**：支持自定义搜索引擎和界面主题
* **Tor 支持**：可通过 Tor 网络使用以增强匿名性

## 部署

1. 拉取镜像并启动：

   ```bash
   docker compose up -d
   ```
2. 浏览器访问 `http://服务器IP:端口` 即可使用

## 默认信息

* 默认端口：`8080`（可在 Compose 文件中修改）
* 默认无需登录，直接使用
